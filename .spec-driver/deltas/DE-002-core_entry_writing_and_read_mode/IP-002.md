## 1. Summary
- **Delta**: DE-002 – Core entry writing and read mode
- **Specs Impacted**: PROD-001
- **Desired Outcome**: A functional `im` that writes timestamped entries to
  daily Markdown files and reads them back via pager — covering all non-editor
  input modes.

## 2. Context & Constraints
- **Current Behaviour**: `im` parses flags and detects input mode but writes
  nothing (diagnostic stubs from DE-001).
- **Target Behaviour**: FR-001–FR-007, FR-009, FR-011, FR-012.
- **Dependencies**: DE-001 complete (merged). Packages: `config`, `cli`, `cmd/im`.
- **Constraints**: Append-only files. < 100ms inline mode. Single binary.
  No editor mode (deferred to DE-003).
