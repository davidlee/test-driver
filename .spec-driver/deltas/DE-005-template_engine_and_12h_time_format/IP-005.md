## 1. Summary
- **Delta**: DE-005 — Template engine and 12h time format
- **Specs Impacted**: PROD-001, SPEC-001
- **Desired Outcome**: Daily files created from configurable templates with dynamic
  variables; frontmatter `updated_at` patched on append; 12h time format option.

- **Target Behaviour**: FR-014 (template engine), FR-015 (dynamic vars + updated_at), FR-016 (12h time).
- **Dependencies**: None (DE-003, DE-004 completed).
- **Constraints**: One new dependency (go-nanoid/v2). Default template must produce identical output.
